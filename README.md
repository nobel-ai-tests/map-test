### Modern Map Features (`modern_map.html`)

The modernized US data map offers extensive customization and interactivity:

#### 📊 Data & Spectrum Control
- **Custom Data Input**: Support for JSON, CSV, or simple space-separated text (e.g., `NY 10`, `California 5`, or `CA - California`).
- **Dynamic Color Spectrum**: Configure low-value, high-value, and zero-value colors, and adjust the number of spectrum steps (2 to 9).
- **Flexible Naming**: Toggle between full state names and abbreviations for labels.

#### 🖋️ Advanced Typography
- **Global & Individual Styling**: Set a global font size or manually override the font size for each individual state.
- **Visual Polish**: Customize label color, font weight, and font style (Italic/Normal).
- **Readability Enhancements**: Adjustable label outline (glow) size ensures text is legible against any background.

#### 🗺️ Interactive Exploration
- **Smooth Zoom & Pan**: Fluid interaction to focus on specific regions.
- **Smart Layering**: States with data are automatically drawn above those with zero values to maintain visual priority.
- **Background Customization**: Choose any color for the map canvas.

#### 📤 High-Resolution Export
- **One-Click PNG Download**: Capture a high-quality (3x scale) image of the map exactly as it appears in your browser, preserving all zoom, data, and styling adjustments.
